my projects!


Creating a 3D graphic scene using Aframe.
I was tasked with transforming a 2D photograph or painting into a captivating 3D
scene using Aframe. Throughout the creation of my scene, I had the creative freedom
to incorporate a diverse mix of 2D and 3D graphic elements, textual elements,
panoramic images, and environments. This project allowed me to explore the synergy
between different mediums, giving life to a unique and immersive experience.

Javascript and Web Practice.
My assignment involved crafting code for each function, ensuring that it stored and
processed data while generating the specified output. This coding challenge
demanded the incorporation of conditionals, loops, functions, and objects to successfully
complete the project.

chatbot.
In our assignment, we were tasked with developing a chatbot using Javascript, Node.js,
and the OpenAI API , alongside utilizing additional frameworks and languages as
deemed appropriate. My team, consisting of four members, was required to collectively
craft a chatbot that, at the very least, assumed a distinctive personality, area of knowledge,
or perspective to respond to user queries. Beyond this foundational requirement, we were
encouraged to unleash our creativity. Throughout this collaborative project, we were
designated into teams, and the code was submitted as a cohesive team effort, with all
members receiving the same grade.
chess game.
Our Chess Group aimed to develop a chess game with basic computer AI logic, prioritizing
the integration of CSS, HTML, and JavaScript for an interactive online experience. We crafted
a user-friendly interface, focusing on the chessboard layout, piece design, and aesthetics.
HTML structured the game's web page, CSS enhanced styling for a pleasing UI, and JavaScript
handled dynamic features like user moves and game logic.
equal pay everyday.
I learned HTML, CSS, and JavaScript through the Girls Who Code Summer Immersion
Program in 2020. My task was to create a social activist website. Therefore, I created
a website named "Equal Pay Everyday" which highlights and informs users about the
wage gap between working men and women.
